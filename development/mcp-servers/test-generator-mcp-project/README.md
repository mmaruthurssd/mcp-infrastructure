# Test Generator MCP Project

**Status:** Active Development
**Category:** Supporting MCP
**Priority:** Low

## Overview

Automated test generation MCP server for unit tests, integration tests, and coverage analysis. Supports Jest, Mocha, Vitest, and supertest frameworks.

## Key Features

- Unit test generation via AST parsing
- Integration test generation (REST APIs)
- Coverage gap analysis
- Test scenario suggestions
- Multiple framework support

## Tools (4)

1. `generate_unit_tests` - Generate unit tests
2. `generate_integration_tests` - Generate integration tests
3. `analyze_coverage_gaps` - Coverage analysis
4. `suggest_test_scenarios` - Scenario suggestions

Drop-in template: `templates-and-patterns/mcp-server-templates/test-generator-mcp-template/`
