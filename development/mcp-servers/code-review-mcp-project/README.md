# Code Review MCP Project

**Status:** Active Development
**Category:** Supporting MCP
**Priority:** Low

## Overview

Comprehensive code quality analysis MCP server with linting, complexity detection, code smell identification, and technical debt tracking. Supports TypeScript, JavaScript, Python, Java, and Apps Script.

## Key Features

- Static analysis (ESLint, Pylint, Checkstyle)
- Complexity metrics (cyclomatic, cognitive, nesting)
- Code smell detection
- Technical debt tracking
- AI-powered improvement suggestions
- Comprehensive review reports

## Tools (6)

1. `analyze_code_quality` - Linting and static analysis
2. `detect_complexity` - Complexity analysis
3. `find_code_smells` - Anti-pattern detection
4. `track_technical_debt` - Debt management
5. `suggest_improvements` - AI-powered suggestions
6. `generate_review_report` - Comprehensive reports
