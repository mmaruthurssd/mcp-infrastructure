# Changelog

All notable changes to mcp-config-manager will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-11-07

### Added
- Standardized 8-folder project structure
- Template-first development approach
- Comprehensive metadata files (README.md, CHANGELOG.md, EVENT_LOG.md)
- Drop-in template for future workspaces

### Changed
- Migrated from local-instances to development/staging structure
- Updated configuration validation against MCP-DEVELOPMENT-STANDARD.md

### Documentation
- Added project README.md
- Created CHANGELOG.md
- Created EVENT_LOG.md for tracking

## [1.0.0] - 2025-10-26

### Initial Release
- Config sync functionality
- Server registration and unregistration
- Server listing with status
- Configuration validation
- Scope auto-detection (user vs project)
