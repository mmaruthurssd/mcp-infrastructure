# 08 - Archive

**Purpose:** Completed and deprecated items

## Structure

- **goals/** - Archived goals
  - **implemented/** - Successfully completed goals
  - **shelved/** - Cancelled/postponed goals
- **workflows/** - Completed workflows
- **brainstorming/** - Old brainstorming sessions
- **deprecated/** - Deprecated files/code

## Notes

- Include date in folder/file names
- Keep retrospectives with archived goals
- Don't delete - archive for reference
