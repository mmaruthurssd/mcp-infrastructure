# 02 - Goals & Roadmap

**Purpose:** Core project goals and roadmap

## Structure

- **ROADMAP.md** - Project phases, timeline, milestones
- **potential-goals/** - Goals being evaluated
- **selected-goals/** - Active goals being worked on
- **workflow-diagrams/** - Visual workflow diagrams

## Goal Folder Structure

Each selected goal gets its own folder:
```
selected-goals/[goal-name]/
├── goal.md              # Goal description, metrics, status
├── spec/                # Specifications
│   └── specification.md
├── tasks.md             # Task breakdown
└── documentation/       # Goal-specific docs
```
